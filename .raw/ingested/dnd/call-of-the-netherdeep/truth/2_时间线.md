# 前日谭1：Story B的时空悲剧与跨位面因果

## 事件 1.1：叛神殿探险队的分裂与瑟西被萨瑞兹顿力量腐化

### 时间/顺序
StoryB玩家小队在玩《溟渊的呼唤》时，探索叛神殿期间 。

### 地点
巴佐赞的叛神殿（Betrayers' Rise）内部，一个充满诡异螺旋（萨瑞兹顿的标志）与锁链的房间 。

### 参与者
- **吉斯·麦尔伯格（Gies Myrberg）**：人类吟游诗人（Story B 主角，即后来的格莱姆） 。
- **瑟西（Cersei）**：精灵术士（吉斯的队友） 。
- **凯瑟琳（Catherine）**：弓箭手/战士（被分隔至另一队的队友） 。
- **厄沃尼（Ervony）**：圣武士（被分隔至另一队的队友） 。

### 事件经过
1.  StoryB玩家小队在探索叛神殿时，受殿内恶魔魔法影响，地牢布局发生变动，导致小队被迫分裂为两组 。
2.  吉斯与瑟西被分到一组，与队友凯瑟琳和厄沃尼失散 。
3.  吉斯与瑟西误入了一个布满螺旋纹路与锁链的房间，这里的石纹像某种神明在梦里咬人的牙，墙壁仿佛在呼吸 。
4.  瑟西被房间内残留的萨瑞兹顿（叛神）魔法力量感染，精神受到腐化 。
5.  被腐化的瑟西眼神变得空洞，仿佛被抽空了灵魂，她突然对吉斯横刀相向，发起了致命攻击 。
6.  吉斯在慌乱的格挡与反击中，不仅未能唤醒瑟西，反而因出手失误意外杀死了她 。

### 直接结果
- StoryB玩家小队被物理隔绝，力量分散 。
- 瑟西当场死亡，尸体倒在布满螺旋纹路的房间内 。
- 吉斯陷入极度的愧疚与孤立无援的状态，为后续与欧门提斯（Omentis）的交易埋下伏笔 。

## 事件 1.2：绝望的抉择：吉斯误杀挚友瑟西与萨瑞兹顿副手欧门提斯的复活交易

### 时间/顺序
在Story B的时间线中，吉斯与瑟西在叛神殿内被魔法分流后 。

### 地点
叛神殿（Betrayers' Rise）内部，一个布满诡异螺旋纹路与锁链、充满萨瑞兹顿魔法气息的房间 。

### 参与者
- **吉斯·麦尔伯格（Gies Myrberg）**：Story B中的人类吟游诗人（后来的格莱姆）。
- **瑟西（Cersei）**：吉斯的挚友与队友，雅灵术士 。
- **欧门提斯（Omentis）**：萨瑞兹顿的副手，被称为“毁灭之苗”或“梦者” 。

### 事件经过
1. 吉斯与瑟西在探索过程中闯入一个充满诡异螺旋与锁链的房间，瑟西被房间内的叛神魔法感染，对吉斯横刀相向 。
2. 在混乱的战斗与扭曲的魔力影响下，吉斯因误判或失手，亲手给予了瑟西致命一击，导致瑟西倒地身亡 。
3. 就在瑟西死去的瞬间，欧门提斯的声音在吉斯脑海中响起，提出可以复活瑟西，但吉斯必须支付代价 。
4. 欧门提斯提出交易条件：将三祷之坠交给他，并需完成特定任务（背叛另外两名队友） 。
5. 面对挚友的尸体，吉斯在绝望中点头同意了契约，接受了欧门提斯的条件以换取瑟西的复活 。

### 直接结果
- 瑟西因交易达成而获得复活（或复活的契机确立），吉斯因此背负了受制于副手的契约枷锁 。

## 事件 1.3：吟游诗人的反击：吉斯利用诀别遗物特性自杀传送神器与欧门提斯的身份剥夺惩罚

### 时间/顺序
在Story B的时间线中，吉斯误杀瑟西并与欧门提斯达成复活交易之后 。

### 地点
巴佐赞（Bazzoxan）的叛神殿（Betrayers' Rise）内部 。

### 参与者
- **吉斯·麦尔伯格（Gies Myrberg）**：Story B小队的吟游诗人，三祷之坠的持有者 。
- **欧门提斯（Omentis）**：萨瑞兹顿的副手，此时作为契约的主导者 。

### 事件经过
1. 欧门提斯提出交易，承诺复活吉斯误杀的队友瑟西，条件是吉斯交出神器“三祷之坠”，并且背叛另外两名队友 。
2. 吉斯表面答应交易，背叛另外两名队友以确保瑟西复活。但为了不让神器落入邪恶之手，他利用了诀别遗物（Vestige of Divergence）的特性：当持有者死亡时，神器会自动触发应急传送机制转移至他处 。
3. 吉斯在叛神殿内亲手杀死了自己，导致三祷之坠瞬间传送出叛神殿，脱离了欧门提斯的掌控范围 。
4. 欧门提斯因未能夺得神器而翻阅契约条款，宣布吉斯违约，并强制执行了身份剥夺的惩罚 。
5. 欧门提斯将“吉斯”的名字和年轻外貌从吉斯的灵​​魂中剥离并据为己有，只给复活后的吉斯留下了一个苍老的面容和重组后的粗糙名字——“格莱姆（Grime）” 。

### 直接结果
- 圣武士厄沃尼死亡。
- 三祷之坠成功逃离了叛神殿和欧门提斯的控制 。
- 真正的吉斯失去了原本的身份，变成了名为“格莱姆·拜格斯”的老人 。
- 欧门提斯盗用了“吉斯·麦尔伯格”的名字与外貌，开始以此伪装在世间行走 。

## 事件 1.4：时间线的缝补：塞琳娜利用拉克桑信标将幸存者从 Story B 抛射至 Story A

### 时间/顺序
在 Story B 时间线的尾声，吉斯（格莱姆）即将死亡的前夕 。

### 地点
Story B 时间线中的施法现场（涉及拉克桑信标与秘迹魔法的运作场域） 。

### 参与者
- **塞琳娜·埃伦薇丝（Selena Erenves）**：奥法纪元艾欧的大师级研究者，此时已是通过 Aevilux 技术抽离的独立个体 。
- **吉斯·麦尔伯格（Gies Myrberg）**：即后来的格莱姆，离梦人吟游诗人 。

## 事件 2.1：因果律的回收：三祷之坠回归 Story A 翡翠石窟

### 时间/顺序
在Story B时间线的吉斯（Gies）死亡并触发诀别遗物的“应急传送”机制之后 。

### 地点
Story A 时间线中吉高（Jigow）附近的翡翠石窟（Emerald Grotto），具体位于深处的莎罕妮祈祷地（Prayer Site of Sehanine / Area E12） 。

### 参与者
- 吉斯·麦尔伯格（Gies Myrberg）：Story B中的三祷之坠持有者（触发源） 
- 三祷之坠（Jewel of Three Prayers）：阿利克辛的诀别遗物 

### 事件经过
1. 随着吉斯在Story B的死亡，三祷之坠触发了“持有者死亡时自动转移至关联地点”的特殊应急机制 。
2. 坠饰依据其与初代使用者阿利克辛（Alyxian）的深层神性联系，锁定了阿利克辛曾向织月者莎罕妮祈祷的地点作为传送落点 。
3. 三祷之坠跨越了Story B与Story A的平行时空界限，从Story B的叛神殿彻底消失 。
4. 金色的坠饰凭空物质化出现在Story A时间线的翡翠石窟深处，安放在一处充满空气的洞穴中央的水晶基座之上 。
5. 坠饰随即进入“休眠态”（Dormant State），其神力内敛，静静等待下一位英雄的到来 。

### 直接结果
- 三祷之坠成功逃离了Story B中萨瑞兹顿副手（Omentis）的掌控，未落入敌手 。
- 该神器确立了其在Story A时间线中的物理位置，成为现在玩家们冒险的StoryA中的关键任务物品 。

## 事件 2.2：沙梅尔的弃婴：格莱姆与瑟西收养茹迪斯之子霍莉及其童年的红色风暴噩梦

- 霍莉（Holly）：被遗弃的女婴，茹迪斯之子 。

### 事件经过
1. 格莱姆在沙梅尔一家旅馆旁的马厩中发现了一名被遗弃在寒冷冬夜的女婴，当时正值红月茹迪斯耀斑闪耀 。
2. 为了从过去的黑暗记忆中抽离并开启新生活，格莱姆提议并与瑟西共同收养了这个孩子，取名为霍莉 。

4. 随着霍莉在枯木镇长大，她展现出茹迪斯之子的特征（红瞳），并开始频繁做关于“红色风暴”的噩梦 。
5. 霍莉在梦中反复看见一名卷发的男子（阿利克辛）走进风暴，并作为“擢升者”与梦境产生了精神共鸣 。
6. 在瑟西的教导下，年幼的霍莉将梦中的红色风暴与男子画在了画纸上 。
7. 格莱姆使用魔法羽毛笔，以口述日记的形式详细记录了霍莉的成长点滴、噩梦内容以及与红月耀斑的关联 。

### 直接结果
- 确立了霍莉作为“擢升者”与阿利克辛之间的精神链接与同源宿命 。
- 留下了记录霍莉身世与噩梦细节的格莱姆日记，藏于枯木镇家中的地下室 。

### 时间/顺序

### 地点

### 参与者

## 摩达、托雷克与克罗登被沙虫吞入虫腹，在内部遭遇寄生怪物并最终从沙虫底部逃脱

- **霍莉**：NPC，核心容器（代号S-α），格莱姆的养女

- **史蒂夫**：NPC，欧德之手士兵
- **格拉多斯-02**：PC，玩家角色

### 事件经过
1. 格莱姆在轮椅上施放“次元门”法术，将自己从椅子上抛入仪式结界内部，随后在地面爬行并紧紧抱住了法阵中央的霍莉 。
2. 格莱姆在精神层面引导月蚀矿的侵蚀转向自己，导致现实中霍莉身上的月蚀晶刺反向生长，将格莱姆的身体贯穿并钉在半空，格莱姆因此牺牲 。
3. 凯瑟琳冲向仪式外圈，将自己受月蚀矿侵蚀的手臂强行插入红色光幕，抓住了霍莉的脚踝 。
4. 登达隆从高台射出一道暗紫色射线击穿了凯瑟琳的胸膛 。
5. 凯瑟琳在断臂的同时，用尽最后力气将霍莉从结界中强行拽出并推向玩家一侧，随后在向格拉多斯-02托付霍莉后死亡 。
6. 史蒂夫冲上前接住了被甩出的霍莉，并将其护在怀中 。

### 直接结果
- 霍莉成功脱离了仪式法阵并幸存，但身体遭受了严重的月蚀矿侵蚀 。
- 格莱姆与凯瑟琳确认死亡，格莱姆的尸体留在法阵中心成为了仪式的一部分 。
- 红梦密会的最终仪式失去了核心容器（S-α），导致仪式出现波动 。

## 眼魔V试图利用托雷克的无限复活体质将其作为永恒能量电池，托雷克选择向鸦后归还赐福以打破循环

### 时间/顺序
在红梦密会大本营B4遗世间的最终决战期间，在格莱姆的晶花崩裂之后，眼魔V正式启动“样本回收”程序之时 。

### 地点
红梦密会大本营地下B4层“遗世间”（Yi Shi Jian），位于仪式圆台上方悬浮的猩红晶花结构中 。

### 参与者
- **托雷克·铁眉**：玩家角色（PC），鸦后勇士，茹迪斯之子
- **V’Rrildemoth（眼魔V/梦者）**：红梦密会幕后主宰
- **鸦后（The Raven Queen）**：神祇（在精神幻象中出现）
- **史蒂夫**：玩家角色（PC/NPC），在托雷克受困期间由玩家暂时接管控制
- **“过去的托雷克”**：托雷克意识中的自我投射

### 事件经过
1. 眼魔V启动“回收程序”，激活了托雷克左肩胛骨深处植入的提纯月蚀矿碎片，导致猩红晶刺从托雷克体内破体而出 。
2. 托雷克被晶刺与红丝线强行拖入高空，撞入并取代了格莱姆原本所在的晶花位置，成为仪式的新核心组件，全身被接入无数连接V的红色细线 。
3. 托雷克的意识陷入精神试炼，面对代表V之控制的巨大黑影，同时“过去的托雷克”出现并指出V企图利用其“天至日期间无限复活”的特性将其作为永恒能量电池 。
4. 史蒂夫挺身而出挡在队伍前方，暂时接管了托雷克玩家的控制权，誓言在托雷克无法举盾时替他抵挡攻击 。
5. 在精神世界中，托雷克面临抉择：继续维持不死之身并赌队友能撑住，或者向鸦后归还“终点”以彻底封死V的利用路径 。
6. 托雷克选择呼唤鸦后，请求她收回赐予的“不死/无限复活”权能 。
7. 鸦后在幻象中现身，从托雷克灵魂中拔出了象征不死特性的“黑羽楔钉”，将其变回凡人。导致黑影牢笼塌缩，现实中的晶刺因锁孔失效而碎裂 。
8. 托雷克失去了被仪式锁定的特性，从半空中的晶花里滑落，重重摔回地面，重新掌握了身体控制权 。

### 直接结果
- 托雷克失去了“鸦后勇士”在天至日期间无限瞬间复活的能力，变回了会彻底死亡的凡人 。
- 眼魔V将托雷克作为“无限循环电池”的计划彻底失败，连接中断 。
- 作为归还赐福的代价，托雷克的最大生命值永久减少12点，并在天至日结束前获得死灵伤害易伤 。
- 眼魔V被迫将目标转移至格拉多斯-02，试图将其作为备用锚点 。

## 史蒂夫在托雷克被困期间挺身而出，通过植入螺旋之核获得力量并以凡人之躯抵挡眼魔V的攻势

### 时间/顺序
在托雷克·铁眉被月蚀矿晶刺强行拖入高空的仪式晶花中并失去身体控制权之后 。

### 地点
红梦密会大本营最深处，B4 遗世间（仪式大厅）。

### 参与者
- **史蒂夫**：欧德之手士兵，艾琳·博尔顿的弟弟，临时玩家角色 。
- **托雷克·铁眉**：被仪式捕获的鸦后勇士 。
- **眼魔 V (V’Rrildemoth)**：红梦密会幕后主宰 。
- **玩家小队**：格拉多斯-02、Jade、摩达·尼特、克罗登等 。

### 事件经过
1. 托雷克被 V 启动的回收程序捕获，化作猩红流星被钉入空中的晶花内，玩家小队因此失去了前排防御者 。
2. 身负重伤、仅剩极少生命值的史蒂夫在绝境中并未退缩，而是选择接纳植入其胸口的“螺旋之核”的力量，这是一种能透支未来生命以换取当下爆发力的寄生引擎 。
3. 史蒂夫依靠这股力量强行站起，向空中被困的托雷克立下誓言，表示在托雷克夺回身体控制权之前，将由自己代替他举起盾牌守护众人 。
4. 史蒂夫拖着巨剑走到队伍与眼魔 V 之间，脚下踩出回音幻影，以凡人之躯构筑起防线 。
5. 史蒂夫向眼魔 V 发出决斗挑衅，喊出“这一轮，先过我”，强制将敌人的注意力从队友身上引向自己 。

### 直接结果
- 原本控制托雷克的玩家暂时失去对托雷克的控制权，转而开始操控史蒂夫进行战斗 。
- 史蒂夫凭借螺旋之核的强化与自身的意志，成功在托雷克缺席的危急关头接管了战场防御职责，正面抵挡眼魔 V 的攻势 。

## 杰德与摩达·尼特利用言语之力与拉克桑信标碎片解除方尖碑束缚，最终释放了被囚禁的欧门提斯

### 时间/顺序
在击败眼魔V之后，处于至日决战的最后阶段 。

### 地点
红梦密会大本营最深处（B4 遗世间），提仕坦方尖碑前 。

### 参与者
- **杰德（Jade）**：拥有提仕坦血脉与言语之力的玩家角色 。
- **摩达·尼特**：拥有拉克桑信标碎片与秘迹魔法知识的玩家角色 。
- **托雷克、克罗登、格拉多斯-02、史蒂夫**：协助压制仪式回弹的队友 。
- **欧门提斯（以吉斯形象出现）**：被囚禁于此的萨瑞兹顿副手 。

### 事件经过
1. 队伍来到半嵌在地面、被黑铁般半透明锁链缠绕的提仕坦方尖碑前，吉斯（欧门提斯）的影像显现并引导杰德与摩达·尼特进行解封仪式 。
2. 摩达·尼特将拉克桑信标碎片贴近方尖碑底座的金线密集处，利用碎片与方尖碑的共鸣将半虚幻的锁链“定锚”在现实中，防止其漂移 。
3. 托雷克、克罗登、格拉多斯-02与史蒂夫分别站在方尖碑周围的四个金线汇聚节点上，按住节点以压制锁链松动产生的能量回弹 。
4. 杰德利用血脉中的“言语之力”念出短促的古语（“立名”与“起誓”），从语言层面改写现实，使锁链表面出现细密裂纹并开始松动 。
5. 尽管杰德胸口的玉佩发出温暖的光芒试图让她在最后时刻停下，但最终最后一个音节还是被吐出，彻底完成了仪式 。

### 直接结果
- 提仕坦方尖碑上的束缚锁链被解除 。
- 被囚禁的欧门提斯被释放 。
