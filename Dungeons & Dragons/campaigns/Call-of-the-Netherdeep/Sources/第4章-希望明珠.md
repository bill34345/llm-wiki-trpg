# 第4章：希望明珠 The Jewel of Hope

## 概述

角色们抵达大都市安卡瑞尔，探索城市与派系，最终前往溺亡之城凯尔·莫罗。

## 安卡瑞尔 Ank'Harel

| 属性 | 描述 |
|------|------|
| 人口 | 293,500 |
| 领导者 | 贾蒙·萨·欧德（黄铜龙） |
| 法律 | 《欧德法典》 |
| 八个区 | 冲积区、桂冠之路、导引区、山脊区、河流区、沙漠先锋区、印记区、蔽日集市 |

## 三大派系

| 派系 | 描述 |
|------|------|
| 全视联盟 | 学术组织，研究凯尔·莫罗 |
| 红梦秘会 | 隐秘研究组织，研究茹蒂斯 |
| 钴魂书院 | 知识收集组织，反对红梦秘会 |

## 派系任务

- 全视联盟："埋藏之物"故事线
- 红梦秘会："猩红诡计"故事线
- 钴魂书院："知识就是力量"故事线
