# 翡翠石窟 The Emerald Grotto

> [!danger] 第1章地下城
> 水下洞穴系统，功业节总决赛的举办地，也是三祷之坠沉睡之所。

## 基本信息

| 属性 | 描述 |
|------|------|
| 位置 | 吉高东南黑岛最小岛屿 |
| 特性 | 完全水下洞穴 |
| 水下呼吸 | 需要药水（持续1小时） |

## 地点

| 区域 | 名称 | 描述 |
|------|------|------|
| E1 | 石窟入口 | 通往水池的狭窄洞穴 |
| E2 | 海藻丛 | 游泳通过需要力量检定 |
| E3 | 洞穴岔路 | 南北两条路径分岔 |
| E4 | 幽灵草生长地 | 发光藻类，危险肉食海草 |
| E5 | 滑坡 | 山体滑坡堵住部分出口 |
| E6 | 食人鱼巢穴 | 食人鱼集群 |
| E7 | 章鱼花园 | 巨章鱼，深坑 |
| E8 | 激流隧道 | 30尺长，湍急水流 |
| E9 | 死胡同？ | 隐藏出口 |
| E10 | 月鲨巢穴 | **翡翠之眼**所在地，巨鲨守护 |
| E11 | 织月者之路 | 通往秘密房间的通道 |
| E12 | 莎罕妮祈祷地 | **三祷之坠**所在地 |

## 关键事件

- 两队（角色 + 对手）竞赛穿越石窟
- 首先到达E10可获得**翡翠之眼**
- E12发现**三祷之坠**，经历阿利克辛的幻象
