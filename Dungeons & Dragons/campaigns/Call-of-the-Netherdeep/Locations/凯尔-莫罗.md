# 凯尔·莫罗 Cael Morrow

> [!danger] 溺亡之城
> 安卡瑞尔地下的古代城市沉没遗址，第5章探索地点。

## 基本信息

| 属性 | 描述 |
|------|------|
| 别名 | 溺亡之城 |
| 原名 | 已失传（被格乌什摧毁） |
| 位置 | 安卡瑞尔地下 |
| 状态 | 被水淹没 |

## 历史

- 原为精灵城市凯尔·莫罗
- 被格乌什的长矛摧毁，陷入地下
- 城市淹没在巨大蓄水池中
- 蓄水池是绿洲水源的重要组成部分

## 重要性

在凯尔·莫罗的深处，隐藏着通向溟渊的裂缝——这是进入溟渊的唯一入口。
