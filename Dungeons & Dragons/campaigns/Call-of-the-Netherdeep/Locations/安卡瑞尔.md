# 安卡瑞尔 Ank'Harel

> [!success] 希望明珠
> 马凯特大陆上的雄伟绿洲城市，第4章的舞台。

## 基本信息

| 属性 | 描述 |
|------|------|
| 人口 | 293,500 |
| 领导者 | 贾蒙·萨·欧德（远古黄铜龙） |
| 代理大臣 | 格梅舍加 |
| 法律 | 《欧德法典》 |
| 守卫 | 欧德之手 |

## 八区

| 区域 | 描述 |
|------|------|
| 冲积区 | 住宅区，黏土采矿 |
| 桂冠之路 | 住宅区，可通过屋顶穿行 |
| 导引区 | 多个正神神庙，记忆之塔 |
| 山脊区 | 工业区，铁匠铺、造船厂 |
| 河流区 | 农业区，运河水培，生命穹顶 |
| 沙漠先锋区 | 军营地，刀疤会 |
| 印记区 | 大学区，水晶城堡（全视联盟） |
| 蔽日集市 | 市场区，"初蚀"酒馆（红梦秘会） |

## 历史

- 被**毁灭者格乌什**创造（祂的长矛一击摧毁了原来的丛林城市）
- 格乌什的长矛创造了沙漠
- 商人建立安卡瑞尔于绿洲周围
- 绿洲水源来自溟渊

## 派系

| 钴魂书院 | 知识收集，反对红梦秘会 |
