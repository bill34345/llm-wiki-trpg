# 吉高 Jigow

> [!info] 第1章地点
> 位于乔哈斯北部的沿海村庄，主要居民为地精和半兽人。

## 基本信息

| 属性 | 描述 |
|------|------|
| 位置 | 乔哈斯北部，翡翠湾和伊弗隆河岸边 |
| 统治 | 长老委员会（地精长老科尔布·卡兹 + 兽人长老乌什鲁） |
| 军事 | 曙光守卫驻守 |
| 特遣官 | 德思·米莉姆（卓尔） |

## 区域划分

| 区域 | 名称 | 描述 |
|------|------|------|
| 乱巷 | Jumble | 中心区，文化发源地，道路错综复杂 |
| 糜水港 | | 主要码头区域 |
| 洼路 | Wetwalks | 建在高架上的房子，靠近湿地沼泽 |

## 功业节 Festival of Merit

吉高最受欢迎的活动，年度庆典，包含7个比赛项目：
- J1: 乱巷最好的馅饼
- J2: 一命通迷宫
- J3: 伊弗隆河弄潮
- J4: 扳手腕挑战
- J5: 洼路收割赛
- J6: 放牧天背龟
- J7: 押韵谜语

## 重要地点

| 地点 | 描述 |
|------|------|

| 手忙脚乱 (J9) | 王朝前哨，曙光守卫驻扎 |
