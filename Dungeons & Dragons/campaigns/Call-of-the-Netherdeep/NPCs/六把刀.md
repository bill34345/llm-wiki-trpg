# 六把刀 Six Knives

> [!warning] 路边劫匪
> "路边匪帮"首领，号称"六把刀"。

## 基本信息

| 属性 | 描述 |
|------|------|
| 真名 | 波斯特拉克 |
| 阵营 | 中立邪恶 |
| 职业 | 强盗首领 Bandit Captain |
| 武器 | 六把匕首（刮骨、冷光、怨恨、猛戳、穿刺、挠瘙） |

## 描述

波斯特拉克，又称"六把刀"，是"路边匪帮"的首领。他头发斑白，骑着一只威风凛凛的黑色战马。他无时无刻不随身携带着六把匕首，并给它们命名：
- 刮骨
- 冷光
- 怨恨
- 猛戳
- 穿刺
- 挠瘙

## 遭遇：路边劫匪

"六把刀"宣称自己是克灵收税员，要求每名角色上交25gp的"道路基础设施税"。

如果角色交钱，他会微笑着脱帽致敬，然后带队离开。

如果角色拒绝，他会让强盗们发起攻击。任意一名强盗（包括六把刀）损失一半生命值后会在下一轮尝试逃跑。
