# 埃洛伊西亚·特尔凡 Aloysia Telfan

> [!warning] 红梦秘会特工
> 红梦秘会的精灵特工，入门者 (Occult Initiate)。

## 基本信息

| 属性 | 描述 |
|------|------|
| 种族 | 精灵 (Elf) |
| 阵营 | 守序邪恶 |
| 身份 | 红梦秘会入门者 |
| 位置 | 巴佐赞 → 安卡瑞尔 |

## 背景

埃洛伊西亚是红梦秘会派驻在巴佐赞的特工。她因为试图贿赂极光守卫的士兵保护她而被谴责。她声称自己打算寻找叛神殿内的一处古老圣地。

## 在第3章

在巴佐赞，埃洛伊西亚尝试接近角色们，提供关于红梦秘会的信息。她被描述为：

- 穿着猩红色长袍的瘦弱人影
- 守序邪恶的精灵秘会入门者
- 渴望知识和权力

## 在第4章

埃洛伊西亚在安卡瑞尔带领角色们前往"初蚀"酒馆，介绍他们给红梦秘会。她是角色们接触红梦秘会的引路人。
