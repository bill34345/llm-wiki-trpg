# D&D 5e 骰子系统参考

## 属性调整值

| 属性值 | 调整值 |
|--------|--------|
| 1 | -5 |
| 2-3 | -4 |
| 4-5 | -3 |
| 6-7 | -2 |
| 8-9 | -1 |
| 10-11 | +0 |
| 12-13 | +1 |
| 14-15 | +2 |
| 16-17 | +3 |
| 18-19 | +4 |
| 20 | +5 |

## 难度等级 (DC)

| 难度 | DC | 描述 |
|------|-----|------|
| 非常简单 | 5 | 无需投掷 |
| 简单 | 10 | 有才能的角色可应付 |
| 中等 | 15 | 需要专业技能 |
| 困难 | 20 | 只有专家能成功 |
| 非常困难 | 25 | 大师级 |
| 几乎不可能 | 30 | 传奇级 |

## 攻击判定

```
d20 + 调整值 + 熟练加成（如果熟练）
```

## 优势 / 劣势

- **优势**：投2d20，取较高者
- **劣势**：投2d20，取较低者
