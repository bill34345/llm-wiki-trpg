# 茹蒂斯 Ruidus

> [!abstract] 厄兆之月
> 艾桑维亚较小的红月亮，迷信与恐惧的对象。

## 基本信息

| 属性 | 描述 |
|------|------|
| 类型 | 较小的卫星/红月 |
| 周期 | 每六个月循环一次 |
| 行为 | 比卡萨更加诡异，有时明亮，有时投下阴影，有时完全消失 |

## 民间传说

关于茹蒂斯的迷信在艾桑维亚很普遍：

| 迷信 | 描述 |
|------|------|
| 先兆 (Harbinger) | 出生在茹蒂斯满月夜的人注定给他人带来痛苦 |
| 恶意 (Malice) | 研究茹蒂斯的人会制造不幸和灾难 |
| 厄运 (Misadventure) | 茹蒂斯满月时启动的计划注定失败 |

## 真实性质

> [!info] 隐藏的真相
> 文中刻意避免讨论茹蒂斯的实际魔力。任何NPC声称从茹蒂斯获得的力量实际来自于**近神者阿利克辛**。

## 月蚀矿

月蚀矿因颜色与茹蒂斯相似而得名，并非真正来源于月球。
