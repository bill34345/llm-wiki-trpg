# 月蚀矿 Ruidium

> [!danger] 腐化元素
> 一种深红色的腐化矿物，源自近神者阿利克辛扭曲的情绪。

## 起源

月蚀矿起源于**溟渊**。当格乌什的矛刺穿物质位面进入水元素位面时，溟渊诞生。近神者被困在溟渊中，他的负面情绪（悲伤、愤怒、绝望）结晶形成了这种矿物。

## 特性

- 深红色，表面有血一般的纹路
- 光滑油亮，具有不为人知的魔法特性
- 能够渗透并生长到接触的一切事物中
- 与月蚀矿结合的生物可能被腐化

## 月蚀腐化 Ruidium Corruption

接触月蚀矿的生物会面临腐化风险：

1. **体质豁免**（DC 取决于月蚀矿的致命性）
2. 豁免失败 → 提升1级力竭并被腐化
3. 力竭等级每下降/提升1级，受到1d10心灵伤害
4. 最终可能致死，或被月蚀矿晶体从内部长满全身

### 腐化阶段

| 力竭等级 | 肉体迹象 |
|----------|----------|
| 1 | 红色皮疹 |
| 2 | 深红色脉络 |
| 3 | 深红色水泡 |
| 4 | 锐利晶体从身体长出 |
| 5 | 水晶增生物越来越大 |
| 6 | 死亡 |

### 结束腐化

只有**祈愿术**或**神圣干预**才能治愈。
