# 面纱帮 The Veil

> [!warning] 犯罪集团
> "欢迎来到另一边。"

## 概述

面纱帮是一个由间谍、小偷和刺客组成的秘密网络，同时也是安卡瑞尔最大的犯罪集团。它是欧德之手守序体现的混乱对位。

## 基本信息

| 属性 | 描述 |
|------|------|
| 口号 | "欢迎来到另一边。" |
| 性质 | 犯罪集团 |
| 总部 | 无（不断更换） |
| 领袖 | **伊莲娜·哈帕哈里**（混乱邪恶人类刺客 Assassin） |
| 同盟 | 无 |
| 对手 | 欧德之手（死对头） |

## 现状

- 大众对面纱帮所知不多
- 成员身份和个人细节在团体内也保密
- 泄露秘密的人往往在被逮捕前就被发现被谋杀
- 在城市没有持久总部，不断更换基地以掩盖踪迹

## 领袖：伊莲娜·哈帕哈里

20岁，来自马凯特西海岸，混乱邪恶人类刺客 Assassin。她最近除掉了面纱帮的旧首领，现在幕后统治着面纱帮——只有高级成员知道这些。

## 与欧德之手的对立

欧德之手未能阻止面纱帮，两派是死对头。

## 资金来源

**阿蒂玛·圣希文**（幸运奔流赌场老板）偷偷用赌场收入的一半资助面纱帮——如同他家族几代人做的那样。
