# 记忆哨兵 Sentinels of Memory

> [!warning] 狂热记忆派系
> "过去的就让它安息吧。"

## 概述

记忆哨兵是一群狂热分子，他们相信安卡瑞尔地下沉没的城市并不是什么伟大的古代奇迹，而是一座**必须不惜一切代价保持密封的监狱**。

## 基本信息

| 属性 | 描述 |
|------|------|
| 口号 | "过去的就让它安息吧。" |
| 性质 | 狂热秘密组织 |
| 总部 | 记忆之塔（导引区） |
| 领袖 | **守望者特拉斯特**（绝对中立精灵祭司 Priest，艾恩前牧师） |
| 行动负责人 | **守望者拜伦**（守序邪恶侏儒老兵 Veteran） |
| 同盟 | 红梦秘会（脆弱的联盟） |
| 对手 | 全视联盟（死对头） |

## 信念

- 凯尔·莫罗不是伟大遗迹，是**监狱**
- 强大力量想把城市从记忆中抹去
- 必须不惜代价确保不再有人提及这座城市
- 历史上文字记载的缺失证明了这一点

## 与全视联盟的冲突

- 反对联盟挖掘凯尔·莫罗
- 大多数努力是政治性的
- 部分成员参与暴力阻挠行动

## 与红梦秘会的联盟

- 秘会提供资金和代理人
- 哨兵怀疑秘会别有用心
- 谨慎对待秘会

## 在第4章

### 核心成员

- **守望者特拉斯特** - 创立者，艾恩前牧师
- **守望者拜伦** - 行动负责人，渴望亲眼看凯尔·莫罗

### 关键行动：立梅尔·威斯特

[[立梅尔-威斯特]]是记忆哨兵成员，假扮全视联盟教授渗透：

1. 建立"第二入口"假情报
2. 召唤隐形追猎者伏击角色（生命穹顶）
3. 将赤红狂怒戒指放入普罗里克斯背包陷害他
4. 领导探险队时篡改记录

### 守望者拜伦

- 守望者特拉斯特不在时负责大部分哨兵行动
- 对派系政治战略越来越不耐烦
- 开始策划更激进方式阻碍全视联盟
- 暗中渴望亲眼看凯尔·莫罗
