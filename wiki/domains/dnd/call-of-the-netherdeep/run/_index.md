# Run Layer

这个目录只放本团私有 campaign canon，不覆写 official module canon。之后 ingest 带团笔记或真相笔记时，优先按下列二级结构落位。

## 目录规则
- `truth/`：世界观真相、幕后机制、时间线解释的上位设定
- `timeline/`：按时间顺序组织的战役 chronology 与事件流
- `characters/`：本团私有人物状态、身份真相与角色线
- `state/`：当前桌面状态、遗物状态、未回收伏笔等运行态记录

## 后续 ingest 约定
- 新的角色类 campaign canon 默认落到 `run/characters/`
- 新的幕后真相、宇宙规则、历史机制默认落到 `run/truth/`
- 新的剧情顺序、session 流、事件链默认落到 `run/timeline/`
- 新的当前状态、桌面推进、物品归属、待处理伏笔默认落到 `run/state/`
- official canon 页面需要对照本团设定时，只加分层提示和回链，不把 campaign canon 写回 official 页
