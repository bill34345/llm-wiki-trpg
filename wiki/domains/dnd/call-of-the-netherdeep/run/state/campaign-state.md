# Campaign State

这个页面用于记录当前桌面的真实推进状态，而不是原模组 canon。

## 当前状态要点
- 战役已经不再停留在“寻找阿利克辛真相”的阶段，而是进入了远地点天至日后果继续扩散、并且已经外溢到安卡瑞尔地表的阶段
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/霍莉]] 已被从最终仪式中救出，并在后续战线里先重新纳回螺旋之核、再于[[wiki/entities/locations/dnd/call-of-the-netherdeep/柯瑞隆神庙]]亲手吸收三祷之坠外层月蚀矿污染
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/格莱姆]] 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/凯瑟琳]] 已确认死亡
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/托雷克]] 已失去天至日期间的无限复活能力
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/史蒂夫]] 已植入螺旋之核，并在安卡瑞尔沦陷时进一步燃烧未来换取过载级爆发
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/艾琳·博尔顿]] 已在无底洞前亲身拦截灾星使徒并重伤倒下
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/杰德]] 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/摩达·尼特]] 已成为释放欧曼提斯的直接执行者，且杰德在柯瑞隆神庙事件后进一步获得音节 / 真言型升级
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/V]] 的核心计划已被打断，但其造成的工程与伤害并未自动消失
- [[wiki/domains/dnd/call-of-the-netherdeep/run/truth/梦境理论与欧曼提斯|欧曼提斯]] 已被释放，后续危机等级明显抬升
- 队伍已不是通过正常城市入口继续冒险，而是在[[wiki/entities/items/dnd/call-of-the-netherdeep/对调胶囊]]帮助下从无底洞防线强行切入[[wiki/entities/locations/dnd/call-of-the-netherdeep/凯尔-莫罗]]，并已在柯瑞隆神庙把[[wiki/domains/dnd/call-of-the-netherdeep/run/state/三祷之坠-战役状态|三祷之坠]]重新解封回收
- [[wiki/entities/monsters/dnd/call-of-the-netherdeep/底栖魔鱼阿利克辛]] 已明确把三祷之坠视为神给自己的补偿，说明队伍即使完成遗物回收，也并未脱离它的追索线
- 安卡瑞尔三大势力已在公开沦陷压力下短暂联手，说明局势已进入真正的战区化状态

## 当前推进位置
- 当前大弧线已推进到 [[wiki/domains/dnd/call-of-the-netherdeep/run/timeline/再次进入凯尔·莫罗与三祷之坠解封]] 之后
- 战场已经从提仕坦遗迹内部扩散到安卡瑞尔城区、无底洞前线与凯尔·莫罗神庙内部
- 队伍当前的核心问题不再只是“阻止谁完成仪式”，而是“如何在欧曼提斯已释放、城市已沦陷、关键承载者仍未脱离绑定、而三祷之坠又已重新回到队伍手里的情况下继续推进”

## 结构性代价
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/史蒂夫]] 代表凡人身体强行承受螺旋之核的代价
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/霍莉]] 代表核心载体即使被救出，也可能再次成为螺旋之核承载点，甚至继续替队伍亲手承担遗物污染
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/艾琳·博尔顿]] 代表留城势力必须亲自下场承担末日级拦截
- [[wiki/entities/monsters/dnd/call-of-the-netherdeep/灾星使徒]] 代表赤月力量已经能直接回收战场材料并生成更高阶代行者
- [[wiki/entities/monsters/dnd/call-of-the-netherdeep/底栖魔鱼阿利克辛]] 则代表即使遗物已经回收，阿利克辛的悲剧与占有欲也仍会追着队伍继续深入
