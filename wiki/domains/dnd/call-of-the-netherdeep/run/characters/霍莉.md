# 霍莉

在这团真相里，霍莉不是普通被卷入事件的孩子，而是天赋极高的擢升者。她在最终阴谋中承担的不是象征性角色，而是螺旋之核制造与破封工程中不可替代的活体终极过滤器；在更深一层，她还是阿利克辛在现世形成的回声体。

## 她是谁
- 年幼的擢升者
- 与茹迪斯的共鸣极高
- 在安卡瑞尔一度化名为希拉
- 能力强到足以在整套仪式工程里承担终极工序
- 与阿利克辛存在同源回声关系

## 她在阴谋里的位置
### 终极过滤器
霍莉承担的不是一般实验体角色，而是螺旋之核提纯流程中的最终过滤阶段。前序流程可以使用其他受体反复消耗、提炼和抽出结晶，但最后一轮必须由共鸣层级更高的个体完成。

### 活体钥匙的一部分
她不仅是材料，更是“活体钥匙”体系的关键部件。她的价值来自擢升者身份本身，而不是单纯身体条件。

### 阿利克辛回声
这份人物志进一步把她解释成阿利克辛在漫长囚禁与梦境中诞生的回声体。于是她不只是“适合被利用的孩子”，而是本来就与阿利克辛共享同源精神结构的人。

## 她经历了什么
### 养父母与童年
她被格莱姆与瑟西收养，在枯木镇长大。从小就反复做关于红色风暴和卷发男子的梦，说明她与阿利克辛的链接早在主线开始前就已经存在。

### 安卡瑞尔化名与死亡复活
来到安卡瑞尔后，她以“希拉”为名与玩家相遇，在旅店夜袭中被误杀，又被伪装成莉莉娅的格拉多斯-02 复活。复活这件事让她与格拉多斯-02 的关系从照顾者升级成几乎无法被切开的生死羁绊。

### 被掳与实验
她后来在花屋被红梦密会带走，在提仕坦遗迹中以 S-α 的身份被当作最理想的实验体使用。螺旋之核的最终纯化，几乎是直接拿她的生命去完成的。

### 至日获救
最终是格莱姆、凯瑟琳、史蒂夫与玩家共同把她从仪式中心救出来。她的获救不是某个单独角色的胜利，而是几条长期人物线在终局同时收束的结果。

### 无底洞前再次持核现身
但这次获救并没有让她和螺旋之核真正切开。进入安卡瑞尔沦陷后的无底洞防线时，托雷克胸口的鸦憩之坠开启裂隙，霍莉重新出现在战场上，手里握着仍在跳动的螺旋之核，并把它直接重新纳回自己体内。

### 从被救者变成再度上场者
她那时外表短暂恢复成近乎健康的状态，甚至能够撑起护盾、抵抗巨爪，并清醒地对队伍说出“现在换我帮助你们”。这一步让她不再只是被从仪式台上拖下来的受害者，而是再次主动介入终局后续战线的人。

### 柯瑞隆神庙里亲手剥离三祷之坠污染
后来队伍再次进入[[wiki/entities/locations/dnd/call-of-the-netherdeep/柯瑞隆神庙]]时，霍莉主动提出由自己吸收包裹[[wiki/entities/items/dnd/call-of-the-netherdeep/三祷之坠]]的月蚀矿层。她不是在别人清场后上前碰一下遗物，而是在底栖魔鱼阿利克辛不断从门外施压、神像坍塌和神庙灌水同步恶化的情况下，亲手把真正的坠饰从污染壳层里剥离出来。

### 她开始直接感受到阿利克辛的悲伤
解封完成后，她没有表现成单纯的成功者，而是清楚感觉到了阿利克辛那种巨大、混杂着怨恨与失落的悲伤。这说明她与阿利克辛 / 螺旋之核的回声联系不只没有减弱，反而在承担遗物污染后进一步变得可感。

## 她为什么危险
- 越靠近远地点天至日，越容易被外部势力强行利用
- 她的存在证明擢升者不只是接收红梦的人，而是可被仪式工程化利用的核心节点
- 一旦被完整纳入流程，她会把阴谋从理论推向可执行状态
- 她与阿利克辛的同源性使她不仅会被利用，也可能被阿利克辛本人试图留住
- 即使已经被救出，她仍可能再次把螺旋之核纳回体内，说明这层绑定不是一次手术或战斗就能切断的

## 当前状态
- 幸存，但身体遭受严重月蚀矿侵蚀
- 在无底洞突入阶段一度把重新出现的螺旋之核直接融回体内
- 后来又在柯瑞隆神庙主动吸收包裹三祷之坠的月蚀矿层，替整个队伍完成了解封工序里最危险的部分
- 能短暂维持行动与护持，但这更像强行承受诅咒后的回光返照，而非真正康复
- 暂时仍由玩家团队保护
- 是战后最需要被保护、同时也最容易继续被旧真相追上的角色之一
- 已能直接感知底栖魔鱼阿利克辛的巨大悲伤，说明她仍深度处在那条回声链路里

## 她与其他人的关系
- 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/凯瑟琳]] 形成前后工序关系：凯瑟琳承担中间过滤，霍莉承担最终过滤
- 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/托雷克]] 同属被工程化利用的关键载体，但用途不同；而且她是在托雷克胸口裂隙开启后重新回到战场的
- 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/格拉多斯-02]] 的关系从复活开始被重新绑定
- 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/史蒂夫]] 一同构成“凡人承担代价”与“核心载体再度承载”的两条终局代价线
- 与[[wiki/domains/dnd/call-of-the-netherdeep/run/state/三祷之坠-战役状态|三祷之坠]]的关系已经推进到“亲手把遗物从月蚀矿污染中剥离出来”的层级
- 与 [[wiki/entities/monsters/dnd/call-of-the-netherdeep/底栖魔鱼阿利克辛]] 的联系不再只是同源设定，而是已经能在遗物解封后直接承受它的悲伤回响
- 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/哑女希拉]] 仅共享名字，不是同一人
- 她的上位解释来自 [[wiki/domains/dnd/call-of-the-netherdeep/run/truth/茹迪斯之子与擢升者]]

## 她在这团主线里的作用
- 解释为什么阴谋方需要“高共鸣个体”而不只是矿石和装置
- 解释螺旋之核的终局工序为什么必须绑定特定角色
- 把擢升者从设定标签变成真正可操作的仪式变量
- 让阿利克辛的痛苦与现世代价真正有了活着的承受者
- 进一步说明“救出霍莉”不等于“螺旋之核问题结束”，因为她本人仍可能再次成为承载点
