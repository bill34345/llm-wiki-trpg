# 欧曼提斯

欧曼提斯是本团里贯穿梦境操控、身份盗用与妖精荒野污染链条的幕后操盘者。这个名字在很多页面里频繁出现，因此即使他的完整上位真相主要放在 run truth 层，也需要一个稳定的人物入口页来承接月刷丛林、假九头蛇与芬迪尔部族相关引用。

## 他是谁
- 能跨梦境与多条人物线布局的幕后存在
- 常通过伪装、交易与精神干预推进计划
- 与[[wiki/entities/locations/dnd/call-of-the-netherdeep/月刷丛林]]、[[wiki/entities/monsters/dnd/call-of-the-netherdeep/假九头蛇]]和妖精荒野污染扩散密切相关
- 在当前知识层中应视作 campaign-canon 的高层真相人物

## 他在当前可复用层面的作用
### 月刷丛林污染链的幕后施力者
在组织与区域层面，欧曼提斯最重要的可复用作用，是把祈愿者皮克精异常化、芬迪尔部族受压迫历史与假九头蛇相关污染接到同一条幕后干预链上。

### 梦境与身份误导的来源
多个 run 页面都把他作为梦境干预者和身份盗用者来处理。因此在 canonical 查询层里，他至少需要被稳定标记为“许多异常事件背后的操盘者”，而不是继续只作为某个 run 真相页标题中的名字存在。

## 为什么他重要
- 他为月刷丛林与假九头蛇线提供统一幕后解释
- 他让芬迪尔部族与祈愿者皮克精的冲突多出上位操盘层
- 他是频繁被引用但此前缺少承接页的关键人物节点
