# 人物志与关系网

## 摘要
这份来源不是单纯的人名索引，而是把本团《Call of the Netherdeep》里最关键的人物真相、身份错位、关系网络、当前动机与终局后状态集中写成了一份角色总档。它的价值不在于给每个人补一句设定，而在于把此前已经散落在 `run/truth/`、`run/timeline/` 与部分人物页里的信息，重新整理成“这个人到底是谁、经历了什么、现在处在什么位置、与谁互相牵制”的可查询角色层。

## 这份来源实际补强了什么
### 1. 把玩家角色从“真相变量”补成完整人物档案
此前 run 人物页已经指出几个玩家角色在阴谋结构中的位置，但这份来源进一步补全了他们的个人起源、专属道具、阶段性死亡与复活、以及当前行动目标：
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/杰德]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/托雷克]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/克罗登]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/摩达·尼特]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/格拉多斯-02]]

### 2. 把身份错位链与养成链补到人物层
这份来源特别重要的一点，是把很多此前只在时间线中提到过的身份错位链补成了人物档案：
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/莉莉娅·格拉汉姆]] 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/格拉多斯-02]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/霍莉]] 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/哑女希拉]]
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/格莱姆]] 与欧曼提斯盗名伪装链
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/艾尔敦]] 与欧曼提斯老者伪装链
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/威尔比]] 与 official canon 的 [[wiki/entities/npcs/dnd/call-of-the-netherdeep/登达隆]]

### 3. 把终局存活者与后果承受者补成后天至日状态入口
这份来源明确了至日决战后仍在运行的人物后果，尤其是：
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/霍莉]] 的极度虚弱与昏迷状态
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/史蒂夫]] 的螺旋之核植入后遗症
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/托雷克]] 失去不死赐福后的凡人化
- [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/杰德]] 与 [[wiki/domains/dnd/call-of-the-netherdeep/run/characters/摩达·尼特]] 作为欧曼提斯释放者的后续负担

## 这份来源能直接回答的问题
- 杰德、托雷克、克罗登、摩达·尼特、格拉多斯-02 现在各自的主线负担是什么？
- 霍莉与阿利克辛、凯瑟琳、格莱姆、格拉多斯-02 之间的关系链怎么串？
- 为什么“莉莉娅”“希拉”“艾尔敦”这些名字在这团里不能直接按单一角色理解？
- 谁在至日决战后仍然活着，但已经带着结构性后果？
- 哪些角色应该进 run 人物层，哪些仍然只保留 official canon 页？

- 阿利克辛、贾米尔、切询、普罗里克斯、对手小队等 official 角色，本页只记录这份来源如何重新解释他们与玩家的关系，不替代各自 official canon 页
