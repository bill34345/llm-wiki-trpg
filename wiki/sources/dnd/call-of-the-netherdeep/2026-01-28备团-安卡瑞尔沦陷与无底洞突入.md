# 2026-01-28备团：安卡瑞尔沦陷与无底洞突入

## 摘要
这份备团把“提仕坦决战之后的安卡瑞尔”第一次真正写成了可跑的灾难现场。它不再只说后果扩散，而是把神殿外的红色裂光、城市平民孵化成怪物、钴魂书院与全视联盟的战场接力、欧德之手在无底洞前的绝望防线、史蒂夫以未来换力量的爆发，以及霍莉带着螺旋之核重新现身的瞬间，压成了一整段从街头逃亡到坠入凯尔·莫罗前夜的连续 run arc。

## 这份来源补强了什么
### 1. 把安卡瑞尔后果扩散从抽象状态推进成具体战场
来源明确写出了城内灾变不是“局势恶化”四个字，而是：
- 地表裂开并喷出猩红能量
- 平民与研究者在红雾中孵化成[[wiki/entities/monsters/dnd/call-of-the-netherdeep/星之子]]
- 钴魂书院、全视联盟与欧德之手分别在不同节点为队伍开路
- [[wiki/entities/locations/dnd/call-of-the-netherdeep/凯尔-莫罗的无底洞]] 前已经变成最后防线

### 2. 把史蒂夫的螺旋之核代价推到“燃烧未来”层级
这份来源让[[wiki/domains/dnd/call-of-the-netherdeep/run/characters/史蒂夫]]不再只是“极不稳定的植入者”。他在无底洞前线主动要求螺旋之核拿走自己的未来，短暂换来极端过载的爆发，并以此把队伍直接送进最后的防线内侧。

### 3. 把无底洞突入从入口动作升级成独立事件弧线
这里不只是“队伍继续往下走”，而是包含了：
- 钴魂书院在街头的第一轮掩护
- 全视联盟战斗法师的火力覆盖与转移决定
- 欧德之手的倒计时式崩线
- [[wiki/entities/npcs/dnd/call-of-the-netherdeep/贾米尔·阿利西亚]]、[[wiki/entities/npcs/dnd/call-of-the-netherdeep/詹姆斯-克里昂]] 与 [[wiki/entities/npcs/dnd/call-of-the-netherdeep/阿拉克里托斯]] 在无底洞前的联合防守
- [[wiki/entities/items/dnd/call-of-the-netherdeep/对调胶囊]] 作为把队伍送入凯尔·莫罗的临时方案

### 4. 把两个新高价值节点压成稳定入口
这份来源直接建立了两个后续很可能继续被引用的节点：
- [[wiki/entities/items/dnd/call-of-the-netherdeep/对调胶囊]]
- [[wiki/entities/monsters/dnd/call-of-the-netherdeep/灾星使徒]]
